# OneByOne

Professional 1:1 Image Cropper. Fast, secure, and precise.

## Features
- **1:1 Precise Cropping**: Optimized for profile pictures, icons, and social media.
- **Advanced Tools**: Zoom, Rotate, Flip, and Fit-to-Width/Height.
- **Custom Resolution**: Export up to 5000px square images.
- **Smart Compression**: Define your target file size (KB) and the engine handles the rest.
- **Format Support**: Export as JPEG (optimized) or Lossless PNG.
- **Privacy First**: All processing happens in your browser. No images are uploaded to any server.

## Tech Stack
- React 18
- Vite
- Tailwind CSS
- Lucide React
- Cropper.js (via react-cropper)
- Vitest & React Testing Library

## Getting Started
1. Install dependencies: `npm install`
2. Start development server: `npm run dev`
3. Run tests: `npm run test`
4. Build for production: `npm run build`
